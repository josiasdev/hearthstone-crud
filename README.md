# HearthStone Card Manager

Projeto desenvolvido em Angular como avaliação Front-end. O sistema permite o gerenciamento (CRUD) de cartas do jogo HearthStone, armazenando os dados localmente no navegador (`localStorage`) e possuindo filtros de busca avançados.

## 🛠️ Tecnologias Utilizadas
* Angular 
* TypeScript
* Less (Pré-processador CSS)
* SSR (Server-Side Rendering) tratado para persistência de dados.

## 🚀 Como compilar e rodar o projeto

1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. Clone este repositório:
   ```
   git clone https://github.com/josiasdev/hearthstone-crud
   ```
3. Acesse a pasta do projeto:
   ```
   cd hearthstone-crud
   ```

4. Instale as dependências:
   ```
   npm install
   ```
5. Execute o servidor de desenvolvimento:
   ```
   ng serve
   ```
   ```
   npm start
   ```
6. Abra o navegador e acesse 
    
   ```
   http://localhost:4200/
   ```
